# ShoppingMall
##购物商城管理端后台

##页面模块
### 公共模块
#### 1、 商户注册页面
#### 2、 商户登录页面(登录后根据商户类型，展示不同的页面)
#### 3、 我的信息页（点击后展示我的账户信息)

### 管理员模块
#### 1、商户接入审批页(对商户的注册信息进行审批)
#### 2、商户列表页(展示商户矩阵列表, 点击后弹窗展示商户详情, 支持删除商户操作)
#### 3、订单列表页(支持根据时间段及订单状态进行过滤)
#### 4、销量统计页(展示商城订单总量、总金额, 并已扇形图形式展示各商户的订单数量及金额分布)

### 商户模块
#### 1、商品列表页（展示商品列表， 支持新增、删除商品，点击后跳转至商品详情页，支持编辑修改商品信息)
#### 2、订单管理页（支持根据时间段及订单状态进行过滤）

### 快递公司模块
#### 1、订单页（展示需要派送的订单，支持接收订单、拒收订单、完成订单操作)
